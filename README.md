---------------------INSTALACÃO----------------------------
- IMPORTANTE: My-Utils é um codigo que fiz para vincular um script no outro, sem ter que ficar sempre
- usando as mesmas importacões a cada vez que fosse usar. Deixe sempre na pasta raiz da aplicacao.

-Abra a pasta de pré-requisitos
-Comece pela instalacao-basica.sh
-Instale os requisitos-python-check
-Depois rode os requisitos-finais.py
-Por ultimo, rode o Debug.py para verificar possiveis falhas.

#A versão final da aplicacao terá todos esses processos unificados, mas por enquanto deixei assim,
ajustes serão feitos no codigo final ainda, então se der algum bug, é bem mais facil de resolver
estando dividido :P


----------------MODO DE USAR------------------------------
- Abra a pasta de Recon no seu terminal
- python3 "www.seudominio.com" ou IP
- Os resultados irão para uma pasta "Resultados"
- Abra a pasta analise para rodar o Analyzer.
- phyton3 Analyzer.py
- Quando você coloca o mesmo dominio do resultado anterior, ele já puxa todos os dados coletados e filtrados para fazer a analise.

-----------------CONCLUSÃO----------------------------------

Com esses resultados, você tem uma lista de dominios ativos e seus servicos rodando neles, para iniciar seus testes de seguranca.
Nenhuma forma ilegal de obter dados foi utilizada, então embora faca algum ruído no servidor do alvo, nenhuma lei está sendo infringida pois são usados apenas fontes abertas para obter esses dados :)

Futuramente, esse projeto ainda vai crescer muito, pois pretendo automatizar até mesmo a fase de testes de vulnerabilidades, fazendo qualquer cacador de recompensas(como eu) ganhar dias de trabalho bracal.
